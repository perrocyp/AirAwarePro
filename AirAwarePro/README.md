# AirAwarePro

## Description
AirAwarePro is an Android application developed by the MECOM student club. This app provides real-time weather information along with the amount of fine dust (in AQI, Air Quality Index) present in the air. It aims to help users stay informed about air quality and take necessary precautions.

## Installation

### Prerequisites
- Android device running version 5.0 (Lollipop) or higher.

### Steps
1. Download the APK file from the [release page](https://github.com/perrocyp/AirAwarePro/releases).
2. Enable installation from unknown sources in your device settings.
3. Open the downloaded APK file and follow the on-screen instructions to install.

## Usage

### Running the App
1. Open AirAwarePro on your Android device.
2. View the main dashboard for current weather and fine dust levels.
3. Click locations other than default to view data in different regions.

### Examples
- Viewing current weather and fine dust levels.
- Checking historical fine dust data.

## Configuration

### Environment Variables
- `API_KEY`: API key for the weather and fine dust data provider. Set this in the app settings.
